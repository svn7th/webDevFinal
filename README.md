# ما هو المشروع؟
المشروع عبارة عن تطبيق لمساعدة الطلاب وزيادة انتاجيتهم


# مميزات المشروع:
1. اضافة مهام ومتابعتها
2. اضافة عادات ومتابعتها
3. التطبيق يقدم مصادر تعليمية متنوعة


# كيف يعمل التطبيق؟
فكرة التطبيق تعتمد على تطبيق مفهوم SPA (Single Page Application)
